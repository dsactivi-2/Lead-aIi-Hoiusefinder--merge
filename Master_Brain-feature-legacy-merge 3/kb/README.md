# Knowledge Base (KB)

Regeln:
- Keine Copy/Paste-Mischung verschiedener Vendor-Dokus in einem Doc.
- Vendor-Inhalte bleiben getrennt nach `kb/vendors/<vendor>/...`
- Unsere Interpretation/Implementierung kommt in `kb/internal/...`
- Jede Aussage mit "Fakt" muss eine Quelle/Verweis haben (source_url oder refs).
- Konflikte werden als `ops/OPEN_QUESTIONS.md` Eintrag erfasst (mit Datum + Links).
