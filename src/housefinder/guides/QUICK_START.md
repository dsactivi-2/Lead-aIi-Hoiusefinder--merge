# Quick Start - Erste Schritte

> In 5 Minuten startklar!

---

## 1. Anmelden

<!-- Screenshot einfügen -->
![Login Screenshot](assets/screenshot_login.png)

2. Gib deine E-Mail ein
3. Gib dein Passwort ein
4. Klicke "Anmelden"
