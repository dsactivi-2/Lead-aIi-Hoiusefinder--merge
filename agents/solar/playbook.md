# Solar Playbook

## Opening
- Confirm consent and call purpose
- Ask for time and permission to continue

## Discovery
- Segment (B2B/B2C), PLZ, consumption, costs
- Property type, roof type, rough area
- Interest in storage or wallbox

## Hard Stops
- No consent (B2C) -> end, set DNC if requested
- DNC request -> set DNC and end

## Next Step
- If qualified: offer/consultation scheduling
- If not: callback or close
